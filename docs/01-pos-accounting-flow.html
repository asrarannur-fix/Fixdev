🚀 POS & Accounting Module Flow

Vertical Slice Completed • Database • API • Frontend

💰 POS Module (Point of Sale)
1
Buka Shift Kasir
Input saldo awal → POST /api/pos/shifts/open
pos_shifts INSERT
2
Scan & Tambah Item ke Keranjang
Barcode → Produk muncul di cart → qty & diskon
products SELECT ← UI
3
Checkout (Create Sale)
POST /api/pos/sales → Server atomic:
INSERT pos_transactions
UPDATE product_stock
INSERT stock_movements
INSERT journal_entries
INSERT journal_lines
INSERT audit_logs
4
Tutup Shift
POST /api/pos/shifts/close → Hitung selisih kas
pos_shifts UPDATE • fraud_alert jika selisih
5
Void / Refund
POST /api/pos/sales/:id/void → Kembalikan stok & jurnal balik
pos_transactions UPDATE • stock_movements • journal_lines
📊 Accounting Module
1
Chart of Accounts (COA)
GET /api/accounting/accounts → Tabel rekening: Aset, Liabilitas, Ekuitas, Pendapatan, Beban
coa_accounts SELECT/INSERT/UPDATE
2
Jurnal Umum (Double-Entry)
POST /api/accounting/journal → Validasi debit = kredit
journal_entries INSERT • journal_lines INSERT • trigger balance sync
3
Buku Kas Operasional
POST /api/accounting/cash → Otomatis buat jurnal kas ↔ akun lawan
journal_entries + journal_lines (via cash tx)
4
Trial Balance
GET /api/accounting/trial-balance → Cek keseimbangan semua akun
coa_accounts SUM(debit-credit)
5
Profit & Loss Report
GET /api/accounting/profit-and-loss → Akumulasi pendapatan - beban
journal_lines SUM by account type • net profit
Built with PostgreSQL (Supabase) • Express.js • React 19 + TypeScript
