# FIXDEV Agent Skill

AI-assisted development skill for the FIXDEV ERP SaaS system.

## Context
Multi-tenant ERP system for Komputer Makassar Service.
Tech stack: React + Vite + TypeScript, Express backend, PostgreSQL (Supabase).

## Modules
- **Services (Servis)**: Service ticket management, technician workflow, QC, handover
- **POS (Kasir)**: Point of Sale, shift management, transactions
- **Inventory (Stok)**: Products, multi-warehouse stock, transfers, purchases
- **Accounting**: COA, journal entries, double-entry bookkeeping
- **HR**: Employees, attendance, payroll, cash advances, commissions
- **CRM**: Customer management, field visits, customer portal
- **WhatsApp**: Message automation, templates, queue management
- **Settings**: Tenant configuration, branding, workflow rules
